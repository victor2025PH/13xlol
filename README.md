## 🚀 快速开始

### 一键安装（Linux/Mac）
```bash
curl -fsSL https://13x.lol/install.sh | bash
```

### Docker 部署
```bash
docker run -d -p 8765:8765 -p 8766:8766 shisanxiang/voice:latest
```

### Windows 安装包
从 [Releases](https://github.com/victor2025PH/13xlol/releases) 下载 `.exe` 安装包，双击运行。

### 从源码构建
```bash
git clone https://github.com/victor2025PH/13xlol.git
cd 13xlol
pip install -r requirements.txt
python launcher.py
```

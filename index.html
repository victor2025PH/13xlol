你可能想问

吃龙虾前的灵魂拷问，这里都有答案

十三香小龙虾到底是什么？跟 ChatGPT 有什么区别？
ChatGPT 是一个模型，我们是一个平台。十三香小龙虾整合了 13+ AI 大模型（包括 ChatGPT），加上语音引擎、微信自动化、远程桌面、工作流引擎等 63+ 技能，本地私有化部署，数据完全在你手里。简单说：ChatGPT 是一味调料，我们是一整锅十三香小龙虾。
真的完全免费吗？有隐藏收费吗？
软件本身 MIT 开源，100% 免费。你只需要自己的 AI API Key（OpenAI、DeepSeek 等）的调用费用。我们的智能路由能帮你自动选最便宜的模型，实测可省 70% Token 费。服务器可以用自己的电脑或几十块的云服务器。
我不懂代码，能用吗？
能！我们提供 Windows 一键安装包（双击 exe 就行）、Docker 一键部署、以及可视化工作流编辑器（拖拖拽拽就能做自动化）。不过微信自动化和远程桌面等高级功能，建议有基础 IT 知识。我们的 Telegram 社群也随时有人帮你。
微信自动化会被封号吗？
我们内置了三层反风控系统：随机延时模拟人类操作节奏、操作频率智能限制、敏感关键词自动拦截。同时支持"人工审核"模式——AI 起草，你一键确认才发送。风险远低于市面上的协议类工具，因为我们是基于 UI 自动化，不碰微信协议。
支持哪些 AI 模型？能用国产模型吗？
支持 13+ 家 AI 供应商：OpenAI (GPT-4o)、Anthropic (Claude)、Google (Gemini)、DeepSeek、智谱 GLM、通义千问、百度文心、Moonshot、零一万物、Groq、Mistral、Ollama 本地模型等。国产模型全覆盖，还能用 Ollama 跑完全离线的本地模型。
数据安全吗？会不会泄露我的聊天记录？
所有数据都在你自己的服务器上，我们不收集任何用户数据。代码完全开源，你可以审查每一行。AI API 调用直接走你自己的 Key，我们不做中间代理。这是自托管方案和 SaaS 服务的本质区别。
为什么选择十三香小龙虾而不是其他 AI 助手？
一句话概括：13+ 模型自动路由（省钱 70%）、完整微信自动化生态（自动回复+朋友圈+群发+多账号）、AI 远程桌面操控、17 节点可视化工作流、9 大 IM 平台桥接、63+ 内置技能——这些能力在普通 AI 助手中要么没有，要么需要付费。而我们完全开源免费，数据 100% 在你手里。
进化路线图是什么？未来还会有哪些功能？
已完成：核心引擎（2024 Q4）→ 全平台覆盖（2025 Q1）→ 微信帝国（2025 Q3）→ v2.0 品牌升级（2026 Q1）。即将到来：2026 Q2 插件市场（社区技能商店 · 第三方工作流模板 · 一键安装插件生态），2026 Q4 多智能体协作（Agent-to-Agent 通信 · 自主任务分解 · 多龙虾协同作战）。

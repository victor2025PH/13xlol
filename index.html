十三香小龙虾到底是什么？跟 ChatGPT 有什么区别？
ChatGPT 是一个模型，我们是一个平台。十三香小龙虾整合了 13+ AI 大模型（包括 ChatGPT），加上语音引擎、微信自动化、远程桌面、工作流引擎等 63+ 技能，本地私有化部署，数据完全在你手里。简单说：ChatGPT 是一味调料，我们是一整锅十三香小龙虾。
真的完全免费吗？有隐藏收费吗？
软件本身 MIT 开源，100% 免费。你只需要自己的 AI API Key（OpenAI、DeepSeek 等）的调用费用。我们的智能路由能帮你自动选最便宜的模型，实测可省 70% Token 费。服务器可以用自己的电脑或几十块的云服务器。
我不懂代码，能用吗？
能！我们提供 Windows 一键安装包（双击 exe 就行，含离线依赖无需联网），以及可视化工作流编辑器（拖拖拽拽就能做自动化）。首次打开会自动引导配置 AI（推荐智谱 GLM-4-Flash，永久免费）。
微信自动化会被封号吗？
我们内置了三层反风控系统：随机延时模拟人类操作节奏、操作频率智能限制、敏感关键词自动拦截。同时支持"人工审核"模式——AI 起草，你一键确认才发送。风险远低于市面上的协议类工具，因为我们是基于 UI 自动化，不碰微信协议。
支持哪些 AI 模型？能用国产模型吗？
